## 1. BSc Computer Science and Engineering (CSE)

**URL**: https://www.tudelft.nl/en/onderwijs/opleidingen/bachelors/computer-science-and-engineering

**Structure**: 180 EC / 3 years. Numerus fixus (~590 students/year). Q1–Q4 quarter system.

### Year 1 — Compulsory Core

| Code | Title | EC |
|---|---|---|
| CSE1100 | Introduction to Programming | 5 |
| CSE1105 | Collaborative Software Engineering Project | 5 |
| CSE1110 | Software Quality and Testing | 5 |
| CSE1200 | Calculus | 5 |
| CSE1205 | Linear Algebra | 5 |
| CSE1210 | Probability Theory and Statistics | 5 |
| CSE1300 | Reasoning & Logic | 5 |
| CSE1305 | Algorithms and Data Structures | 5 |
| CSE1400 | Computer Organisation | 5 |
| CSE1405 | Computer Networks | 5 |
| CSE1500 | Web & Database Technology | 5 |
| CSE1505 | Information and Data Management | 5 |

Year 1 organises into 5 modules: Engineering; Mathematics; Systems; Algorithms & Formal Methods; Responsible Computer Science.

### Year 2 — Core + Variant Track

**Core**: Algorithm Design, Theoretical Foundations of CS, Concepts of Programming Languages, Introduction to AI, Machine Learning, Computer Security, Operating Systems, Software Engineering Methods.

**Variant tracks** (choose one):
- **Multimedia**: Signal Processing, Image Processing, Multimedia Analysis
- **Data**: Big Data Processing, Data Mining, Computational Intelligence
- **Systems**: Digital Systems, Embedded Software, Operating Systems

**CSE2000 Software Project**: 10-week team project with an external client.

### Year 3

- **Minor** (Q1–Q2, 30 EC): broaden to another TU Delft programme or study abroad.
- **CSE3000 Research Project**: individual 10-week research project with a TU Delft supervisor.
- **5 electives**: e.g., Computer Graphics, Big Data Processing, Deep Learning, Bioinformatics, Software Engineering with AI.

### BSc Themes (for matching applicant expertise)

1. **Programming & Software Engineering** — Java, OOP, design patterns, testing, collaborative development.
2. **Algorithms & Formal Methods** — data structures, complexity, graph algorithms, computability.
3. **Systems** — computer architecture, operating systems, networks, embedded systems.
4. **AI & Data** — introductory AI, machine learning, data management, statistics.
5. **Responsible Computing** — security fundamentals, HCI, ethical AI, societal context.

## 2. MSc Computer Science

**URL**: https://www.tudelft.nl/en/education/programmes/masters/cs/msc-computer-science

**Structure**: 120 EC / 2 years. 5 core courses (Q1) + 2 themes (15 EC each across Q2–Q4) + electives + Master Thesis.

### Core Courses (Q1, 5 EC each)

- **Software Architecture** *(mandatory)*
- **Responsible Computer Science** *(mandatory)*
- *Choose 3 of 6*:
  - Machine and Deep Learning
  - Formal Reasoning about Software
  - 3D Computer Graphics and Animation
  - Security and Cryptography
  - Web-Scale Data Management
  - High Performance Data Networking

### Themes (choose 2 of 9–10)

| Theme | Representative Q2–Q4 Courses |
|---|---|
| **Algorithmics** | Exact and Approximation Algorithms; Graph Algorithms; Combinatorial Optimization |
| **Computer Graphics** | Vision & Graphics; Geometric Computing; Physics-based Rendering |
| **Computer Systems** | Real-time Systems; Modern Computer Architecture (RISC-V); Smart Phone Sensing |
| **Cyber Security** | Systems Security; Network Security; Privacy-Enhancing Technologies |
| **Distributed Systems Engineering** | Distributed Algorithms; Cloud Computing; Blockchain Engineering |
| **Information Management** | Data Modelling; Information Retrieval; Web Science & Engineering |
| **Machine Learning Applications Engineering** | ML Pipelines; Deployment; Evaluation & Testing |
| **Programming Languages** | PL Theory; Type Systems; Implementation |
| **Software Engineering** | Requirements Engineering; Software Architecture; DevOps |
| **Quantum Computer Science** | Quantum Information; Quantum Algorithms; Post-Quantum Crypto |

## 3. MSc Data Science and AI Technology (DSAIT)

**URL**: https://www.tudelft.nl/en/education/programmes/masters/dsait/msc-data-science-and-artificial-intelligence-technology

**Structure**: 120 EC / 2 years. Core (Q1) + 2 themes + research course + electives + thesis.

### Core Courses (Q1, 5 EC each)

- **Data Management and Engineering** — modelling, quality, storage, distribution, NoSQL
- **Machine and Deep Learning** — CNNs, RNNs, Transformers, supervised/unsupervised, evaluation
- **Probabilistic AI and Reasoning** — logic, constraint satisfaction, Bayesian networks, MDP/RL
- **Software Engineering and Testing for AI Systems** — ML evaluation, adversarial attacks, label validation
- **Responsible Data Science and AI Engineering** — agency, robustness, privacy, fairness, accountability
- **Research course (Q4)** — Empirical Research Methods OR Fundamental Research in ML/DL

### Themes (choose 2)

| Theme | Q2–Q4 Courses |
|---|---|
| **Advanced Machine Learning** | Deep Learning; Reinforcement Learning; Bayesian Machine Learning |
| **Human-Centred AI** | Human-Centred Machine Perception; Conversational Agents; Designing HCAI Systems |
| **Data Information Management** | Web-Scale Data Management; Information Retrieval; Web Science & Engineering |
| **Optimisation and Reasoning** | Combinatorial Optimisation; Constraint Reasoning; Search |
| **Probabilistic Decision Making** | Decision Theory; Bayesian Statistics; MDPs and RL |
| **Software Engineering for DS&AI** | Testing AI Systems; MLOps; Responsible AI Engineering |
| **Visual Computing** | Image Processing; Computer Vision; Visual Analytics |
| **Language and Speech Technology** | NLP; Speech Recognition; Dialogue Systems |
| **Bioinformatics** | Computational Biology; Genomics; Protein Structure |
