## OpenReview Venues

### NeurIPS 2025
| Field | Value |
|-------|-------|
| Platform | OpenReview |
| Response granularity | Per-review (global rebuttal removed) |
| Character limit | **10,000 characters** per review response |
| Format | Markdown text only |
| File uploads | Not allowed |
| Links | Not allowed (except AC-mediated code-link scenario) |
| Revised paper upload | Not allowed until camera-ready |
| Visibility timing | Rebuttals revealed to reviewers/ACs **at end of rebuttal window** (not immediately) |
| Discussion phase | Reviewer-author discussion window follows rebuttal window |
| Source | neurips.cc/Conferences/2025/PaperInformation/NeurIPS-FAQ |

### ICML 2025
| Field | Value |
|-------|-------|
| Platform | OpenReview |
| Response granularity | Per-review response |
| Character limit | **5,000 characters** per response |
| Format | Markdown text |
| File uploads | Not allowed during author feedback period |
| Links/URLs | No non-anonymized/personal/shortened URLs |
| Revised paper upload | Not allowed during feedback period |
| Discussion flow | Author rebuttal → reviewer acknowledgement (required) → limited follow-up round |
| Source | icml.cc/Conferences/2025/AuthorInstructions |

### ICLR 2026
| Field | Value |
|-------|-------|
| Platform | OpenReview |
| Response model | Public discussion with author comments and reviewer interaction |
| Per-comment limit | Word limit exists but exact number not confirmed in public docs |
| Number of comments | No limit on number of comments |
| Revised paper | Allowed during discussion period (diff tooling available) |
| Discussion model | Iterative — treat as evolving thread, not terminal message |
| Source | iclr.cc/Conferences/2026/AuthorGuide |

### AAAI-26
| Field | Value |
|-------|-------|
| Platform | OpenReview |
| Response model | **Single rebuttal for all reviews** |
| Character limit | **2,500 characters total** |
| Format | Text with markdown |
| Links/URLs | Disallowed; may be withheld if included |
| File uploads | Not allowed |
| New results | Explicitly disallowed |
| Decision note | Recommendations are not simple score averages; senior-area review uses broader signals including rebuttal/discussion |
| Source | aaai.org/conference/aaai/aaai-26/aaai-26-faq-for-authors-rebuttal/ |

## HotCRP Venues

### OSDI 2025 (USENIX)
| Field | Value |
|-------|-------|
| Platform | HotCRP |
| Response cap | **2,000 words** |
| New experiments/data | **Explicitly disallowed** |
| New promised work | Disallowed |
| Review type | Single-blind |
| Source | usenix.org/conference/osdi25/call-for-papers |

### SOSP 2025
| Field | Value |
|-------|-------|
| Platform | HotCRP |
| Response | Optional |
| Length guidance | Strongly encouraged **under 500 words** |
| Hard cap | Not stated; reviewers not expected to read very long responses |
| Review type | Double-blind |
| Source | sigops.org/s/conferences/sosp/2025/cfp.html |

### ASPLOS 2025
| Field | Value |
|-------|-------|
| Platform | HotCRP |
| Length guidance | Strongly encouraged **under 800 words** |
| Hard cap | Not explicitly stated |
| Attachments | Not allowed in rebuttal |
| New experiments/data | Allowed when major revision is realistically possible |
| Source | asplos-conference.org/asplos2025/cfp/ |

### EuroSys 2025
| Field | Value |
|-------|-------|
| Platform | HotCRP |
| Response cap | **1,000 words** (strict) |
| PDF/figures in response | Disallowed — may be removed if included |
| Format | Text only |
| Source | 2025.eurosys.org/cfp |

### NSDI 2025 (USENIX)

| Review type | Single-blind |
| Source | usenix.org/conference/nsdi25/call-for-papers |

### SIGCOMM 2025
| Field | Value |
